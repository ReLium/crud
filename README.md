# crud
CRUD demo project in Golang

## Usage

- `make build`
- `docker-compose up -d`
- `make generate`
- go to http://127.0.0.1:8080/
- swagger UI http://127.0.0.1:8082/?url=http://127.0.0.1:8080/swagger.yaml
